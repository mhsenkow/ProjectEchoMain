## ğŸ“¦ **Installation**

### Prerequisites
- Node.js 18+ and npm
- Python 3.8+ with PyPDF2 (`pip install PyPDF2`)
- Ollama with at least one model (e.g., `llama3.2:latest`)

### Setup
```bash
# Clone the repository

cd echo-frame

# Install dependencies
npm install

# Build the application
npm run build

# Start the application
npm start
```
